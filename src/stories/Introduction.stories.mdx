# Welcome to react-bootstrap-storybook

This library is custom from the <strong>react-bootstrap</strong> library. 
The changes are not too big but help developers save a lot of time when creating a new web application. 
This is made possible by simplifying the use of components included in the <strong>react-bootstrap</strong> library.
